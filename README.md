# TaskFlow ✅

A clean, fully-featured personal task manager built as a single HTML file — no installs, no accounts, no subscriptions. Just open it and start tracking.

**Live app:**[https://curlysglitchin.github.io/taskflow](https://curlysglitchin.github.io/taskflow/)

-----

## Features

### Task Management

- Create tasks with a name, description, creation date, and optional due date
- Separate **Work** and **Personal** tabs to keep tasks organized
- Active tasks stay at the top — completed tasks move automatically to a dated Completed section
- Tasks grouped by completion date (Today, Yesterday, or full date)

### Priority & Organization

- **Highlight colors** — mark tasks pink (urgent), yellow, blue, green, or purple
- Urgent/highlighted tasks automatically sort to the top of the active list
- **Flag tasks** with context tags: Waiting, Review, Question, or Follow-up
- Tasks sort by urgency → due date → creation date automatically

### Subtasks

- Add subtasks to any task by tapping the expand arrow (▾)
- Each subtask has its own checkbox
- Subtask progress shows as a counter chip on the card (e.g. 📋 2/4)

### Notes & Updates

- Tap 💬 on any card to open the notes panel
- Add timestamped notes/updates to a task over time
- Great for tracking progress on ongoing or rolling tasks

### Recurring Tasks

- Set tasks to repeat: Daily, Weekly, Monthly, or Quarterly
- When you check a recurring task complete, the next occurrence is automatically created with the next due date

### Search & Filters

- Search bar searches task names, descriptions, and notes
- Filter chips: All, 🌸 Urgent, 🚩 Flagged, 📅 Due Soon, ⚠ Overdue, 🔁 Recurring

### Dark Mode

- Toggle between light and dark mode with the 🌙 button
- Preference is saved and remembered between sessions

### Export & Backup

- **Export CSV** — opens in Excel, Google Sheets, or Numbers
- **Export JSON** — full backup of all tasks, subtasks, notes, and settings
- **Restore from backup** — import a previously saved JSON file to restore all your tasks

## How to Use

### In the browser

Just visit [curlyglitch.github.io/taskflow](https://curlyglitch.github.io/taskflow) in any browser.

### Add to iPhone Home Screen

1. Open the link in **Safari**
1. Tap the **Share button** (box with arrow) at the bottom
1. Tap **“Add to Home Screen”**
1. Name it **TaskFlow** and tap **Add**

It will appear on your home screen like a native app icon.

### On Desktop

Open the URL in any modern browser. The modal centers on screen and hover states are fully supported for mouse users.

-----

## Data & Privacy

- All tasks are stored locally in your **browser’s localStorage**
- No data is sent to any server
- No accounts, no tracking, no ads
- Tasks are private to your device and browser — anyone else visiting the URL sees a blank app
- Use the **Export JSON** backup regularly to keep a safe copy of your tasks

> ⚠️ Clearing your browser’s site data will erase your tasks. Export a backup first!
